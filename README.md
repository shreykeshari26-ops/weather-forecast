# 🌦️ Weather Forecast App

A modern, responsive weather forecast web application built using React + Vite that provides real-time weather data and forecasts using the OpenWeather API.

---

## 🚀 Live Demo

👉 [https://your-vercel-link.vercel.app](https://weather-forecast-xi-amber.vercel.app/)

---

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature display
- ☁️ Weather conditions (Clouds, Rain, Clear, etc.)
- 📊 Hourly forecast view
- 📅 Weekly forecast
- 🌬️ Additional metrics:
  - Wind speed
  - Visibility
  - Pressure
  - UV Index
- 🎨 Modern UI with glassmorphism design
- 📱 Responsive layout

---

## 🛠️ Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **API:** OpenWeather API
- **Deployment:** Vercel
